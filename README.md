# RouteSmart

Sistema de optimización de rutas para empresas de entrega.

## Problema
Las empresas pequeñas planifican rutas manualmente generando pérdidas.

## Solución
Sistema web que calcula rutas óptimas utilizando algoritmo Dijkstra.
